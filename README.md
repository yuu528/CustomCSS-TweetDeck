# CustomCSS-TweetDeck
TweetDeck用 Gruvbox風 CSS

## 環境
```
OS:        Windows 10 Pro 20H2 19042.804 x64
Browser:   Firefox 86.0 x64
Addon:     Better TweetDeck 3.10.1
TweetDeck: Release July 25th 2019
```

## 使い方
Better TweetDeckのカスタムCSSに本CSSファイルを適用する。
